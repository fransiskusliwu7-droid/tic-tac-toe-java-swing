# Tic-Tac-Toe Java Swing

## Name
Fransiskus Parulian Liwu

## Description
This project is a simple Tic-Tac-Toe game built using Java Swing with database integration.

## Features
- Login using MySQL database
- Tic-Tac-Toe game
- Win, Lose, Draw statistics
- Score system
- Top 5 Scorers

## Database
MySQL (1 table: players)

## How to Run
1. Run XAMPP
2. Import database using schema.sql
3. Run project in NetBeans

## Demo
This application demonstrates:
- Login validation
- Game interaction
- Database update after each game
- Display of Top 5 scorers
